Privacy Policy

Last updated: June 2026

1. Overview

NarrativePilot AI is an open-source project. This Privacy Policy explains what information is collected when you use the Software and how it is handled. We are committed to protecting your privacy and operating with full transparency.

2. Information We Do Not Collect

The NarrativePilot AI landing page (this website) does not collect, store, or process any personal data. There are no analytics trackers, cookies, sign-up forms, or user accounts. We do not use Google Analytics, Meta Pixel, or any other third-party tracking service.

3. Information Processed Locally by the Software

When you run the NarrativePilot AI agent on your own machine, the following data is processed locally and never transmitted to us:

• Your CMC API key (stored in your local `.env` file, never logged or sent externally)
• Your agent wallet credentials (managed entirely by Trust Wallet Agent Kit, stored encrypted in your OS keychain)
• Trade history and portfolio values (written to `logs/trades.log` on your local machine only)
• Market data fetched from CoinMarketCap API (processed in memory, not stored beyond the current session)

4. Third-Party Services

The Software interacts with the following third-party services when running in mainnet or testnet mode. Their respective privacy policies govern how they handle data:

• CoinMarketCap API — market data provider. See coinmarketcap.com/privacy
• Trust Wallet Agent Kit — transaction signing and wallet management. See trustwallet.com/privacy-policy
• BNB Chain — public blockchain. All on-chain transactions are publicly visible by nature of the technology.

5. GitHub Repository

The source code is hosted publicly on GitHub. GitHub may collect usage data when you visit the repository. See GitHub's Privacy Statement for details.

6. On-Chain Transparency

All trades executed by the agent on BNB Chain are permanently recorded on the public blockchain. Anyone can view the agent wallet's transaction history by searching the wallet address on BscScan. This is an inherent property of public blockchains and not a privacy concern specific to this Software.

7. Children's Privacy

This Software is not intended for use by anyone under the age of 18. We do not knowingly collect any information from minors.

8. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected on this page with an updated date.

9. Contact

If you have any questions about this Privacy Policy, please contact us at mariosalamancaenredes@gmail.com.
